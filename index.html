opengl transformation 1

开始学线性代数了。。。

缩放：
S1 0 0 0 x S1 * x
0 S2 0 0 * y = S2 * y
0 0 S3 0 z S3 * z
0 0 0 1 1 1

位移
1 0 0 T1 x x + T1
0 1 0 T2 * y = y + T2
0 0 0 T3 z z + T3
0 0 0 1 1 1

x y z w
如果表示空间顶点的位置，w为1
如果表示两点的差或其他量，w为0。（自己体会）

贴图的uv坐标：
一张图，可以看成是一个u坐标为0至1的浮点数，v坐标为0至1的浮点数的坐标。
u可以看成x坐标，v可以看成y坐标。

曾经的绘图API只支持正方形的贴图，现在可以用长方形的贴图了。但是，但是，长方形的边还是只能是2的幂次方。

采样的方式：nearest和linear
linear通过加权计算像素的颜色，而nearest直接取最近的像素块的颜色
