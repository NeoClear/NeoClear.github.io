adjacency table
2019-11-16

传统的邻接表。比用vector和map实现效率更高。
vertex[i]记录与点i相邻的点的第一个Edge边。通过遍历获得所有相邻的点。
需要注意的是边的index从1开始，因为0代表着遍历的结束。
